# Food Vendor Ordering System 🍽️

A web-based ordering system designed to help food vendors turn visitors into paying customers with less friction and more professionalism.

Instead of relying on back-and-forth WhatsApp chats, this system allows customers to browse food items, customize their order, and instantly generate a structured order summary.

---

## 🚀 What This Project Does

- Displays food items in a clean, mobile-friendly interface
- Allows customers to customize orders (quantity, protein, extras)
- Calculates total price in real-time as selections are made
- Generates a structured order summary
- Sends order directly to WhatsApp for fast completion

---

## 🧠 Core Idea

Most food vendors lose customers during chat conversations because:
- customers get confused about options
- pricing is unclear
- responses take too long

This system solves that by letting customers:
> see → customize → understand total → order instantly
